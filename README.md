# DevWallah 🛠️

**DevWallah** is a comprehensive, open-source collection of developer tools built with Next.js 14, focusing on speed, privacy, and visual excellence. All processing happens client-side, ensuring no user data ever leaves your browser.

**Live Production URL**: [devwallah.com](https://www.devwallah.com)

---

## ✨ Key Features

-   **🔒 Privacy First**: All tools run 100% in your browser. No data is sent to any server.
-   **⭐ Favorites System**: Quickly access your most-used tools by starring them on the homepage.
-   **🔗 Universal Share**: Compress any tool input/state into a shareable URL for easy collaboration.
-   **🌓 Dark/Light Mode**: A premium, glassmorphism-inspired design system with full theme support.
-   **⚡ Lightning Fast**: Statically optimized for instant page loads and zero-latency interactions.

---

## 🧰 Tools Included (30 Total)

### 📄 Formatters & Parsers
-   **JSON Formatter**: Beautify, minify, and validate JSON.
-   **SQL Formatter**: Standardize SQL queries (PostgreSQL, MySQL, etc.).
-   **XML Formatter**: Clean and format complex XML structures.
-   **URL Parser**: Breakdown URLs into components (query, host, path).
-   **JWT Decoder**: Safely inspect JWT headers and payloads.

### 🔄 Converters
-   **Base64 Encoder/Decoder**: Translate strings/files to Base64.
-   **CSV to JSON**: Convert spreadsheet data into JSON arrays.
-   **JSON to CSV**: Flatten JSON arrays into downloadable CSV files.
-   **JSON to YAML**: Seamless transition between data formats.
-   **JSON to TypeScript**: Generate interface definitions from JSON.
-   **HTML to Markdown**: Convert web snippets to clean Markdown.
-   **Color Converter**: Transform HEX, RGB, HSL, and CMYK.
-   **Timestamp Converter**: Unix timestamps to human-readable dates.
-   **Base64 to Image**: Render images directly from Base64 strings.
-   **Text to Unicode**: Escape characters for code and status messages.

### 🎲 Generators
-   **UUID Generator**: Bulk generate v4 UUIDs.
-   **Password Generator**: Cryptographically secure passphrases.
-   **Lorem Ipsum**: Fast placeholder text generation.
-   **Cron Generator**: Human-friendly cron expression builder.
-   **QR Code**: Generate downloadable QR codes for any text.
-   **Hash Generator**: MD5, SHA-1, SHA-256, and SHA-512 hashes.

### 🛠️ Utilities
-   **Word Counter**: Real-time stats (words, characters, reading time).
-   **Regex Tester**: Live regular expression testing with group extraction.
-   **Markdown Preview**: Side-by-side Markdown editor and previewer.
-   **List Sorter**: Alphabetical or numerical sorting (A-Z, Z-A).
-   **Case Converter**: camelCase, snake_case, PascalCase, kebab-case, etc.

## 🛠️ Tech Stack

-   **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Libraries**: `lz-string` (compression), `dayjs` (dates), `sql-formatter`, `marked` (markdown), `qrcode`.
-   **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites
-   Node.js 18.17 or later

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/kaushik-tripathi/devtool.git
    cd devtool
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running Locally
1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open [http://localhost:3000](http://localhost:3000) in your browser.
